## Description:
### This application is used for planning and posting events. It also helps attendees know how to plan and prepare for attending the event.
## Installation:
### You can use this link, https://shaq-pro-event-planning.herokuapp.com, to access the site and use as intended. If you prefer to work on the code, you can pull this repository and run npm i to ensure you have all packages installed to make the application work. 
## Usage:
### This program is intended to be used by event planners to post events and see you has RSVP'd to their event. It is also to be used by attendees to use the weather portion to help decide whether or not they would like to come as well as how to dress/prepare.
